Recursify Learning

This was my project from my Software Design and Development Class. My aim was to develop an app that can be used by year 9 students to learn about graphing, basic measures of spread and also area and perimeter. The app contained a quiz section where they can answer questions to test their understanding. In addition, I also used the numpy library to graph trigonometric and parabolic equations entered by the user. This was a very fun project as I got to learn about Tkinter library of Python as well as numpy and pyplot.

Arduino Robot

This was a project that I developed to make myself more familiar with Arduino coding. I was using Arduino C to develop a robot that could avoid obstacles, choose the path with the least number of obstacles when moving away and also follow a line. I found it amazing to see how various components have to be coded separately to make a bigger project work. Each Actuator and Sensor had to talk to each other and I felt that my code was bridging them. Detecting Obstacles with ultrasonic sensor was an amazing part of my project because it was my first time experiencing "applying theory into practice" moment. I knew how the ultrasonic sensor calculates the distance between the robot and its surroundings. However, using this knowledge to create an obstacle avoidance robot was awesome. I later added a Reflective Optical Sensor to detect black and white to make it into a line following robot. Seeing my code work really made the experience even more memorable.
